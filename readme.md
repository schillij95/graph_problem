## How to Use the Playground
### Create a Directory Structure:

Copy code
```css
graph_problem/
├── CMakeLists.txt
└── example.cpp
Build the Project:
```
Copy code
```bash
mkdir build
cd build
cmake ..
make
```

This will generate the graph_problem executable in the build directory.
Run the Executable:

```bash
./build/example
```

Enjoy!
